Sobre o Jogo

Ecological Frog é um jogo educativo inspirado nos clássicos arcade, desenvolvido com Python e Pygame. O jogador controla um sapo ecológico que precisa atravessar rios, escapar de perigos e recolher resíduos recicláveis para proteger o ambiente.

Tecnologias Utilizadas

Python

Lógica principal e mecânicas do jogo.

Pygame

Motor gráfico responsável pelas animações, colisões e sons.

HTML + CSS

Criação da interface e apresentação do projeto.
